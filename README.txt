              This was a swift code written app to learn the fundimentals of components and the usage of xcode with swift version 3
              
USAGE:
  The app was developed to eliminate the use of a checklist of to-do items.
  
  A new to-do item can be created and checked off when completed at the main gui interface. The item then can be deleted from the list or a new item can be added.
  
HOW TO USE
  after copying the git source code to your xcode software, you need to then install swift language in the xcode terminal and run version 3.
  
  then the app can be installed to a connected iphone via a cord.
